# DoubleJay Assemblies Demo

Conversion-focused demo website built for **DoubleJay Assemblies**.

## Tech Stack
- Next.js 14 (App Router)
- React 18
- Plain CSS (mobile-first)

## Purpose
This is a prospect demo designed to improve quote requests and direct calls through:
- Clear hero + CTA hierarchy
- Service cards
- Service area visibility
- Trust section
- Frictionless quote form UX

## Run Locally
```bash
npm install
npm run dev
```

Open http://localhost:3000

## Build
```bash
npm run build
npm start
```
